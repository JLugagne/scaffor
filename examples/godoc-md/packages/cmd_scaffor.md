# cmd/scaffor

Command-line entrypoint for the scaffor tool.

## Overview

The `cmd/scaffor` package contains the main entry point for the scaffor CLI application. It is responsible for parsing command-line arguments and delegating to the appropriate command handlers.

## Key Files

- `main.go` — Application entrypoint

## Usage

```bash
scaffor [command] [flags]
```

See the [CLI Commands](../cli-commands.md) documentation for available commands.
