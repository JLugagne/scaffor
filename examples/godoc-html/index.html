Package Overview

internal/scaffor

CLI initialization and root command setup. Exposes the Setup() function that returns a configured CLI runner.

app/commands

Application-level handlers for scaffolding operations. Contains ScaffolderHandler which orchestrates template execution, linting, and testing.

domain

Core domain models: Template, TemplateCommand, FileEvent, and execution options. Defines the contract for all scaffolding operations.

domain/repositories/filesystem

Filesystem repository interface. Abstracts file I/O for testability and multiple implementations.

domain/service

Service interface that defines the contract for scaffolding commands: list, get, execute, lint, test.

inbound/cli/commands

CLI command implementations using Cobra. Maps user commands to scaffolding operations.

inbound/mcp

Model Context Protocol server implementation. Enables scaffor as an MCP tool for Claude and other AI agents.

outbound/filesystem

Concrete filesystem implementation. Provides OS-level file I/O operations.
