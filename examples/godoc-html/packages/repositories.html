repositories/filesystem

Import: github.com/JLugagne/joist/internal/joist/domain/repositories/filesystem

Overview

The repositories/filesystem package provides file system abstractions for the domain layer. It defines interfaces for file I/O operations that allow the scaffolding system to work with different file system implementations.

Interfaces

FileSystem

▶

FileSystem is a combined interface for file system operations.

type FileSystem interface {
	ReadFile
	WriteFile
	MkdirAll
}
