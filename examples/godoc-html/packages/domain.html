domain

Import: github.com/JLugagne/joist/internal/joist/domain

Overview

The domain package contains the core types and entities for the Joist scaffolding tool. It defines templates, commands, variables, and validation logic.

Types

Template

▶

Template represents a scaffolding template with its metadata and commands.

type Template struct {
	Name          string
	Description   string
	Commands      []TemplateCommand
	ShellCommands []ShellCommand
}

TemplateCommand

▶

TemplateCommand defines a single command within a template.

type TemplateCommand struct {
	Command      string
	Description  string
	Variables    []TemplateVariable
	Files        []TemplateFile
	PostCommands []string
	Hint         string
}

TemplateVariable

▶

TemplateVariable represents a variable that can be set during template execution.

type TemplateVariable struct {
	Key         string
	Description string
}

TemplateFile

▶

TemplateFile specifies a file to be created from a template.

type TemplateFile struct {
	Source      string
	Destination string
}

LintError

▶

LintError represents a validation error found in a template manifest.

type LintError struct {
	Command string
	Field   string
	Message string
}

func (e LintError) Error() string

ShellCommand

▶

ShellCommand represents a shell command to run after template execution.

type ShellCommand struct {
	Command string
	Mode    string
}
