Import: github.com/JLugagne/scaffor/internal/scaffor/app/commands

Overview

Package commands provides the application-level handlers for scaffolding operations. It contains the ScaffolderHandler which orchestrates template discovery, execution, validation, and testing. This is the core orchestration layer that coordinates between the domain models and the filesystem.

ScaffolderHandler handles manifest-driven scaffolding operations.

ScaffolderHandler is the main orchestrator that implements the ScaffolderCommands service interface. It coordinates template discovery, execution, validation, and testing.
