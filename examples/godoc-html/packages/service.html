internal/scaffor/domain/service

Import: github.com/JLugagne/scaffor/internal/scaffor/domain/service

Overview

Package service defines the ScaffolderCommands interface, which establishes the contract for scaffolding operations. This interface decouples the CLI and MCP inbound adapters from the concrete implementation, allowing different implementations to serve different protocols.

Interfaces

ScaffolderCommands

▶

ScaffolderCommands defines the contract for all scaffolding operations.

type ScaffolderCommands interface {
    ListTemplates(ctx context.Context) ([]Template, error)
    GetTemplate(ctx context.Context, templateName string) (Template, error)
    Execute(ctx context.Context, templateName, commandName string, params map[string]string, opts ExecuteOptions) ([]FileEvent, error)
    Lint(ctx context.Context, templateName string, templateDir string) []LintError
    Test(ctx context.Context, templateName string) error
}

Methods:

  • ListTemplates — Discover all available templates
  • GetTemplate — Load a single template by name
  • Execute — Run a template command with parameters
  • Lint — Validate a template manifest
  • Test — Run a template's test suite

This interface is implemented by ScaffolderHandler in the app/commands package and is consumed by CLI commands and the MCP server.
