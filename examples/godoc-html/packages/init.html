Overview

Package scaffor provides CLI initialization and the root command setup. It wires together all the layers of the scaffor application (filesystem, handlers, commands) and exposes the Setup() function that returns a configured CLI runner.

Types

Runner

▶

Runner is a function that executes the CLI logic.

type Runner func(ctx context.Context, args []string) error

A Runner takes a context and command-line arguments and executes the CLI, returning an error if the execution fails.

Functions

Setup()

▶

Setup initializes and returns a configured CLI runner.

func Setup() Runner

Setup performs dependency injection, wiring the filesystem implementation, scaffolder handler, and all CLI commands. It returns a Runner function that executes the Cobra CLI with the given context and arguments.

Typical workflow:

  • scaffor list — List available templates
  • scaffor doc <template> — Read a template's documentation
  • scaffor doc <template> <command> — See a specific command's variables
  • scaffor execute <template> <command> --set Key=Value — Execute a template command
