internal/scaffor/outbound/filesystem

Import: github.com/JLugagne/scaffor/internal/scaffor/outbound/filesystem

Overview

Package filesystem provides a concrete implementation of the FileSystem repository interface using the operating system's file I/O. This is the outbound adapter that translates repository methods into actual filesystem operations.
