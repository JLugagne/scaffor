internal/scaffor/domain/repositories/filesystem

Import: github.com/JLugagne/scaffor/internal/scaffor/domain/repositories/filesystem

Overview

Package filesystem defines the FileSystem repository interface. This interface abstracts all file I/O operations, enabling testability through mock implementations and allowing multiple concrete implementations for different storage backends.

Interfaces

FileSystem

▶

FileSystem defines the interface for file system operations.

type FileSystem interface {
    ReadFile(ctx context.Context, path string) ([]byte, error)
    WriteFile(ctx context.Context, path string, data []byte) error
    MkdirAll(ctx context.Context, path string) error
}

Methods:

  • ReadFile — Read the contents of a file as a byte slice
  • WriteFile — Write data to a file, creating or overwriting as needed
  • MkdirAll — Create all parent directories as needed (like mkdir -p)

All operations accept a context for cancellation and timeout support.
