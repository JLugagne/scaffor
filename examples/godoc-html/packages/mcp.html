internal/scaffor/inbound/mcp

Import: github.com/JLugagne/scaffor/internal/scaffor/inbound/mcp

Overview

Package mcp implements the Model Context Protocol (MCP) server for scaffor. This allows scaffor to be exposed as an MCP tool for Claude, Cline, and other AI agents. It also provides session tracking functionality to log tool calls and file events for debugging and auditing.

Types

ToolCall

▶

ToolCall represents a single MCP tool invocation.

type ToolCall struct {
    Timestamp  time.Time          // When the tool was called
    Tool       string             // Tool name
    Parameters map[string]any     // Input parameters
    FileEvents []FileEvent        // Files affected
}

FileEvent

▶

FileEvent represents a file operation within an MCP session.

type FileEvent struct {
    Path   string // File path
    Action string // Action performed (created, overwritten, skipped)
}

Session

▶

Session tracks all tool calls and file events in an MCP session.

type Session struct {
    path string        // Path to session log file
    mu   sync.Mutex    // Protects concurrent access
}

Sessions log every tool invocation and the files affected, enabling audit trails and debugging of AI-driven scaffolding operations.
