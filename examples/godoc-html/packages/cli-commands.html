internal/scaffor/inbound/cli/commands

Import: github.com/JLugagne/scaffor/internal/scaffor/inbound/cli/commands

Overview

Package commands implements the Cobra CLI commands for scaffor. It provides the user-facing command-line interface, mapping user inputs to the underlying ScaffolderCommands service interface. Each function returns a *cobra.Command ready to be added to the CLI router.
